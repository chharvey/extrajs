Static format

  • format(date: Date, format: string): string
    • Defined in Date.class.ts:94

    Format a date, using PHP-based formatting options.

    The following options are supported (with examples):

    • 'Y-m-d' : '2017-08-05'
    • 'j M Y' : '5 Aug 2017'
    • 'd F Y' : '05 August 2017'
    • 'l, j F, Y' : 'Friday, 5 August, 2017'
    • 'j M' : '5 Aug'
    • 'M Y' : 'Aug 2017'
    • 'M j' : 'Aug 5'
    • 'M j, Y' : 'Aug 5, 2017'
    • 'F j, Y' : 'August 5, 2017'
    • 'M' : 'Aug'
    • 'H:i' : '21:33'
    • 'g:ia' : '9:33pm'
    • 'default' : '2017-08-06T01:33:00.000Z' (Date#toISOString)
    see

    http://php.net/manual/en/function.date.php

    Parameters

    • date: Date

      the date to format

    • format: string

      one of the enumerated options listed in the description

    Returns string

    a string representing the given date in the given format
